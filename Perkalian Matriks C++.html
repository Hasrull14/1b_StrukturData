Konsep Perkalian Matriks

Jika kita memiliki dua matriks:

Mengapa Ada 3 Perulangan?

Kita gunakan 3 perulangan karena:

  1. Perulangan pertama ( for(i=0; i<2; i++) )

    • Mengiterasi baris dari matriks hasil CCC.

  2. Perulangan kedua ( for(j=0; j<2; j++) )

    • Mengiterasi kolom dari matriks hasil CCC.

  3. Perulangan ketiga ( for(k=0; k<2; k++) )

    • Menghitung total penjumlahan dari perkalian baris dan kolom.

Kode perkalian matriks:

cpp
int hasil_kali[2][2] = {0}; // Inisialisasi array hasil dengan nol
for(i = 0; i < 2; i++) {       // Loop untuk baris hasil
    for(j = 0; j < 2; j++) {   // Loop untuk kolom hasil
        hasil_kali[i][j] = 0;  // Inisialisasi sebelum menjumlahkan
        for(k = 0; k < 2; k++) { // Loop untuk perkalian elemen
            hasil_kali[i][j] += matrix_a[i][k] * matrix_b[k][j];
