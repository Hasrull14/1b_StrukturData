Apa itu Stack?

Stack itu struktur data yang elemennya ditambah dan dihapus hanya dari satu sisi, yaitu bagian atas (top). Jadi, prinsip kerjanya LIFO (Last In, First Out):

Yang masuk terakhir, keluar duluan.

Operasi Dasar Stack

  1. Init (Inisialisasi):
    top = 0; → tandanya stack masih kosong.

  2. PUSH (Menambah Elemen):
    Tambahkan data ke atas stack jika belum penuh.

  3. POP (Menghapus Elemen):
    Hapus data paling atas (terakhir dimasukkan). Jangan dilakukan kalau stack kosong.

  4. FULL:
    Mengecek apakah stack sudah penuh → top == max.

  5. EMPTY:
    Mengecek apakah stack kosong → top == 0.

  6. CLEAR:
    Mengosongkan stack → top = 0;

Deklarasi Stack di C++

Ada 2 cara:

cpp
#define MaxS 10
char isi[MaxS];
int top;

atau dengan struct:

cpp
#define MaxS 10
struct Stack {
    char isi[MaxS];
    int top;
};

Contoh Program Singkat (dari modul)

cpp
void push(stack &s, char data) {
    if(s.top < maxs) {
        s.top++;
        s.isi[s.top] = data;
    } else {
        cout << "Stack Penuh!";
    }
}

cpp
void pop(stack &s, char &hasil) {
    if(s.top != 0) {
        hasil = s.isi[s.top];
        s.top--;
    } else {
        cout << "Stack Kosong!";
    }
}

Contoh Program Stack dengan Menu di C++

cpp
#include <iostream>
using namespace std;

#define maxs 5

struct Stack {
    char isi[maxs];
    int top;
};

// Inisialisasi stack
void init(Stack &s) {
    s.top = -1;
}

// Menambahkan elemen ke stack (PUSH)
void push(Stack &s, char data) {
    if (s.top < maxs - 1) {
        s.top++;
        s.isi[s.top] = data;
        cout << "Karakter '" << data << "' ditambahkan ke stack.\n";
    } else {
        cout << "Stack penuh!\n";
    }
}

// Menghapus elemen dari stack (POP)
void pop(Stack &s) {
    if (s.top >= 0) {
        cout << "Karakter '" << s.isi[s.top] << "' diambil dari stack.\n";
        s.top--;
    } else {
        cout << "Stack kosong!\n";
    }
}

// Menampilkan isi stack
void cetak(Stack s) {
    if (s.top >= 0) {
        cout << "Isi stack: ";
        for (int i = 0; i <= s.top; i++) {
            cout << s.isi[i] << " ";
        }
        cout << endl;
    } else {
        cout << "Stack kosong!\n";
    }
}

int main() {
    Stack s;
    init(s);
    int pilih;
    char huruf;
    do {
        cout << "\nPROGRAM STACK\n";
        cout << "====================\n";
        cout << "[1] Tambah Isi Stack\n";
        cout << "[2] Ambil Isi Stack\n";
        cout << "[3] Tampilkan Isi Stack\n";
        cout << "[4] Keluar\n";
        cout << "Pilih menu: ";
        cin >> pilih;

        switch (pilih) {
            case 1:
                cout << "Masukkan karakter: ";
                cin >> huruf;
                push(s, huruf);
                break;
            case 2:
                pop(s);
                break;
            case 3:
                cetak(s);
                break;
            case 4:
                cout << "Keluar dari program.\n";
                break;
            default:
                cout << "Pilihan tidak valid!\n";
        }
    } while (pilih != 4);

    return 0;
}

Penjelasan Singkat

  • Struct Stack: Menyimpan data dan penunjuk top (posisi teratas).

  • push: Menambah karakter ke stack, dicek dulu apakah penuh.

  • pop: Menghapus karakter paling atas, dicek dulu apakah kosong.

  • cetak: Menampilkan isi stack dari bawah ke atas.

  • main: Program utama dengan menu interaktif.

Kenapa tidak pakai int maxs = 5; langsung?

Jawabannya: karena int maxs = 5; adalah variabel biasa, dan tidak bisa dipakai untuk menentukan ukuran array statis.

Contoh ini:

cpp
int maxs = 5;
char isi[maxs];  // ERROR!

Di C++ standar (khususnya yang lama), ukuran array statis harus berupa konstanta tetap (constant expression), bukan variabel.

Solusinya:

1. Pakai #define:

cpp
#define maxs 5
char isi[maxs];  // BISA

2. Atau pakai const int (jika standar C++ mendukung const expression):

cpp
const int maxs = 5;
char isi[maxs];  // BISA di sebagian compiler modern

Kalau kamu pakai compiler C++ modern (seperti C++11 ke atas), const int bisa menggantikan #define dan lebih disarankan karena:

  • Bisa dicek tipe datanya

  • Lebih aman (respect scoping / jangkauan variabel)

  • Bisa di-debug

Kesimpulan:

  • #define dipakai karena klasik, simpel, dan kompatibel ke semua compiler.

  • int maxs = 5; gak bisa buat array karena bukan constant expression.

  • Kalau kamu sudah pakai C++11 ke atas, lebih baik ganti ke:

    cpp
    const int maxs = 5;

Fungsi top:

top dipakai untuk:

  1. Menunjukkan posisi elemen paling atas di stack

  2. Sebagai indikator jumlah elemen yang ada di dalam stack

  3. Di-increment saat PUSH, dan di-decrement saat POP

cpp
top = -1;

Lalu kita PUSH satu data:

cpp
top++;  // jadi 0
isi[top] = 'A';  // simpan di posisi 0

PUSH lagi:

cpp
top++;  // jadi 1
isi[top] = 'B';  // simpan di posisi 1

POP:

cpp
char dihapus = isi[top];  // ambil 'B'
top--;  // jadi 0 lagi

Kenapa bukan top = 0 dari awal?

Sebenarnya bisa aja. Ada 2 pendekatan umum:

  • top = -1 (paling umum, mulai dari -1)

  • top = 0 (mulai dari 0, tapi push langsung pakai isi[top], baru top++ setelahnya)

Modul kamu pakai pendekatan top = 0, artinya isi dimulai dari index 1. Tapi secara umum di C++, lebih disarankan pakai top = -1 dan mulai dari index 0, biar konsisten sama array.

Bedanya Gimana?

1. Versi Modul (Manual Stack - Old School)

  • Pakai struct, array, dan top

  • Semua operasi (push/pop) kita buat sendiri

  • Lebih cocok buat belajar dasar struktur data

  • Bisa disesuaikan banget, misal: stack untuk ekspresi matematika, karakter, atau logika custom

Contoh:

cpp
struct Stack {
    char isi[10];
    int top;
};

2. Versi Modern (STL Stack - C++ Standard Template Library)

  • Tinggal pakai #include <stack>

  • Semua udah disediakan: push(), pop(), top(), empty()

  • Lebih efisien dan clean

  • Cocok untuk proyek nyata atau lomba koding

Contoh:

cpp
#include <iostream>
#include <stack>
using namespace std;

int main() {
    stack<char> s;
    s.push('A');
    s.push('B');
    cout << "Top: " << s.top() << endl;  // B
    s.pop();
    cout << "Top after pop: " << s.top() << endl;  // A
}

Jadi, kenapa di kampus diajarin versi manual?

Karena tujuannya bukan cuma bisa make stack, tapi ngerti cara kerjanya. Jadi kamu ngerti:

  • Kenapa stack bisa LIFO

  • Gimana cara simpan datanya

  • Gimana top bekerja di balik layar

Kesimpulan:

  • Versi modul = manual, bagus buat belajar

  • Versi STL = modern, bagus buat praktik nyata

  • Kamu udah masuk tahap siap pakai STL — keren!
